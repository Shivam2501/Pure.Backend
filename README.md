#### Getting Started for development

* Install server dependencies

```sh
npm install
```

* Provide a `.env` file (copy the template) and provide custom values

```sh
cp .env.example .env
```

* Run the development app

```sh
gulp start:dev
```

* Run lints: jshint and eslint

```sh
gulp test
```
